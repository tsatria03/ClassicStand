New in 1.0.

You can now buy ice to make a pitcher of lemonade.
Made it so temperatures are no longer randomized throughout the day. The game now uses a dynamic temperature system that scales based on the current day, and time of day. Each day defines a temperature range, which increases the more the game progresses.
Made it so the game's clock time folows the day time pattern, instead of randomizing the time each day. It will also denote weather the time is set to A-M or P-M when reading the time in the game's main menu.
Made it so the day and level systems are now separated. Each day contains 6 levels, representing time blocks, such as early morning, early afternoon, mid afternoon, early evening, midnight, and late night. After reaching level 6, the level resets back to 1, and the day increases by 1.
The pricing system has  now been reworked to scale with both the current day and level. This allows for gradual and realistic economic inflation. This system encourages long-term strategy, high-stakes resource management, and optimized pricing.
